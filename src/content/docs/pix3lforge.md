# Pix3lForge

Pix3lForge is a professional cross-platform desktop image editor built with C++ and Qt 6.9. It provides advanced image manipulation and editing capabilities as a native desktop application for Windows, macOS, and Linux.

**Repository:** [github.com/Pix3ltools-lab/pix3lforge](https://github.com/Pix3ltools-lab/pix3lforge)

## Tech Stack

- **Language:** C++
- **UI Framework:** Qt 6.9
- **Platform:** Windows, macOS, Linux

## Building from Source

### Prerequisites

- Qt 6.9 or later
- CMake 3.16 or later
- C++17 compatible compiler (GCC, Clang, MSVC)

### Build

```bash
git clone https://github.com/Pix3ltools-lab/pix3lforge.git
cd pix3lforge

mkdir build && cd build
cmake ..
cmake --build . --config Release
```

For more details, refer to the [README](https://github.com/Pix3ltools-lab/pix3lforge#readme) in the repository.
